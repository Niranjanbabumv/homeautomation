Voice controlled home automation powered by Watson Speech-to-Text

This file is part of a demo project for voice controlled home automation using IBM Watson Speech to Text API. You can speak one of the supported commands after clicking on the 'Speak' button below. The converted text commands will be published to PubNub network on a particular channel. Accompanying webpage subscribed to PubNub network on the same channel will receive the commands and simualte actions in a SVG image of a home.

The text returned by Watson Speech to Text service will be shown below the 'Speak' button. Any feedback received from the smart home simulator web page will be shown in the Status window.

Supported commands are:

  • Turn on Living Room light. / Turn off Living Room light.
  • Turn on Kitchen light. / Turn off Kitchen light.
  • Turn on Bedroom light. / Turn off Bedroom light.
  • Turn on Portico light. / Turn off Portico light.
  • Turn on Children room light. / Turn off Children room light.

    • You can use these commands in a sentence or a request like structure. For example, you can say "Can you please turn on the bedroom light?".
